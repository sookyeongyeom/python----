# if문

> 일치 확인

> 기본


<br>

> if문과 결합

```python
dabjeongneo = 2

while dabjeongneo != 1:
    dabjeongneo = input("1번이랑 2번 중에 뭐가 좋아? : ")
```

> break

```python

        print("흐흠 그럴 줄 알았어! *^-^*")
```

<br>

> continue

```python

    dabjeongneo = input("1번이랑 2번 중에 뭐가 좋아? : ")
```


```python
import keyboard

white=" "
position = " ■"
print(position)
while True:
    if keyboard.read_key()=="d":
        white += " "
        print(white + position)
    elif keyboard.read_key()=="a":
        white = white.replace(" ","",1)
        print(white + position)
```
